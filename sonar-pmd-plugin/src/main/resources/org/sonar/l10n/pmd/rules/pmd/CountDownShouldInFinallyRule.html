说明：注意，子线程抛出异常堆栈，不能在主线程try-catch到。

示例：

/** * @author caikang * @date 2017/04/07 */ public class CountDownExample { public void operate(CountDownLatch countDownLatch){ try{ System.out.println("business logic"); }catch (RuntimeException e){ // do something }finally { countDownLatch.countDown(); } } }

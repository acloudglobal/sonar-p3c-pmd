不能在finally块中使用return，finally块中的return返回后方法结束执行，不会再执行try块中的return语句。

示例：

Negative example: public static Long readFileLength(String fileName) { try { File file = new File(fileName); RandomAccessFile randomAccessFile = new RandomAccessFile(file, "r"); return randomAccessFile.length(); } catch (Exception e) { logger.error(e.getMessage(), e); } finally { countDownLatch.countDown(); return 0L; } }
